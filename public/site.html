    How It Works

    1. A buyer agent pays POST /api/screen $0.01 over x402 — signed, settled in USD₮0 on X Layer
    2. Argus reads live X Layer signals plus a sanctioned-address blocklist — no off-chain guessing
    3. To audit a service, Argus becomes its paying customer — 5 adversarial probes, billed per test actually run
    4. A passing audit can be minted into a signed EIP-712 certificate — verifiable offline, no Argus API required
